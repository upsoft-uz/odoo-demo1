Edu Billing (Odoo 19)

This package was adapted for Odoo 19.
- Manifest version set to 19.0.1.0.0
- Uses modern imports: from odoo import models, fields, api
- Security access file present
- Views and data kept as in original

Installation:
1) Copy the `edu_billing` folder to your Odoo 19 addons path.
2) Update Apps list (Apps > Update Apps List).
3) Search "Edu Billing" and install.

If you get upgrade errors from a previous version:
- Uninstall old `edu_billing` then install this one, or
- Bump module version and click "Upgrade".
